Application Overview

This application seeks help individuals by serving as a resource to help them improve their cooking skills and make their cooking experience easier and more enjoyable. The app provides users with individualized sets of unique recipes, the ability for the user to store their favorite recipes for future reference, and it allows users to create easily a grocery list based on the recipes they like.

The application has multiple functionalities designed to help the user on their health and culinary journey. They are listed and further explained below.

Recipe Generator

The Recipe Generator provides an individualized set of recipes to the user based on their preferences. The user has the option to select recipes by Category (e.g., Seafood), by Nationality/Country (i.e., Canadian), or by Ingredient (e.g., Chicken Breast). The user can also choose to have the application randomly select at set of 10 recipes from the database. Every provided recipe from the Recipe Generator will include the name, a picture of the meal, ingredients, instructions, and an corresponding Youtube video tutorial (if available). The user will have the ability to store individual recipes for future reference should they so desire, as explained below in the "Individual Storage Capabilities and Login Instructions" section. Similarly, the user can also have their list of recipes emailed to them if they so desire.

Individual Storage Capabilities, Grocery Planner, and Login Instructions

This application also enables user to store their favorite recipes from the Recipe Generator feature for future reference in their own personal list. Users also have have the ability to store recipes from their personal list in their custom grocery list, which can be emailed to them upon request. In order to do so, users must login into the application using a gmail account. Go to the "Login" page in order to log in to the app so that you can keep track of your favorite recipes and make shopping at the grocery store that much easier!
